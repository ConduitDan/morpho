Detailed Description

Matrices are a purely numerical collection type oriented toward linear algebra. Elements are stored in column-major format, i.e. [ 1 2 ] [ 3 4 ] is stored ( 1, 3, 2, 4 ) in memory. This is for compatibility with standard linear algebra packages
